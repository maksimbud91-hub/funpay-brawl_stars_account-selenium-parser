# FunPay Lots Scraper

Автоматизированный парсер товарных лотов с платформы FunPay на базе Selenium и Pandas. Скрипт собирает актуальную информацию о предложениях в выбранной категории, анализирует профили продавцов и сохраняет результаты в структурированную таблицу Excel для последующего анализа или мониторинга цен.

## Основной функционал

- Умное ожидание элементов: Использование WebDriverWait (Explicit Waits) вместо жестких пауз time.sleep, что оптимизирует скорость работы и предотвращает падения при медленной загрузке страниц.
- Высокая отказоустойчивость: Независимая обработка полей каждого лота через блоки try/except гарантирует, что парсер соберет всю доступную информацию, даже если у отдельных продавцов скрыты никнеймы, отсутствует рейтинг или время пребывания на сайте.
- Безопасное управление ресурсами: Использование конструкции try/finally для гарантированного закрытия сессии WebDriver и очистки системных процессов браузера.
- Автоматический экспорт данных: Быстрое преобразование собранного массива данных в объект Pandas DataFrame и выгрузка в формат .xlsx.

## Стек технологий

- Python 3.x — основной язык разработки.
- Selenium WebDriver — автоматизация действий в браузере и динамический парсинг.
- Pandas — обработка данных и формирование табличной структуры.
- OpenPyXL — движок для генерации файлов Excel.

## Установка и запуск

1. Установите необходимые зависимости проекта:
   `bash
   pip install selenium pandas openpyxl
2. Убедитесь, что на компьютере установлен браузер Google Chrome
  (Selenium WebDriver подберет нужную версию драйвера автоматически).
3. Запустите скрипт: python main.py
